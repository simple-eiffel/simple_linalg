Overview

simple_linalg is a production-ready linear algebra library for Eiffel. It provides vectors (2D, 3D, n-dimensional), matrices with proven numerical stability, and advanced algorithms like LU decomposition, condition number estimation, and eigenvalue computation.

Part of the Simple Eiffel ecosystem.

Features

  • Vector specialization (2D/3D) with generic n-dimensional support
  • Matrix operations with column-major storage
  • LU decomposition with partial pivoting (Tier 3 expert access)
  • Condition number validation (κ ≤ 1e12 threshold)
  • SCOOP-safe immutable result objects
  • 3-tier API (Simple/Advanced/Expert)
  • Design by Contract throughout
  • Void-safe implementation
