Result Classes

EIGENVALUE_RESULT

Immutable result from eigenvalue computation.

Features

  • eigenvalue: REAL_64 — Computed eigenvalue
  • eigenvector: VECTOR_N — Associated eigenvector
  • iterations: INTEGER — Iterations performed
  • tolerance_achieved: REAL_64 — Final tolerance reached
  • is_converged: BOOLEAN — Convergence status

LU_DECOMPOSITION

LU factors from matrix decomposition.

Features

  • l_factor: MATRIX — Lower triangular factor
  • u_factor: MATRIX — Upper triangular factor
  • pivot_row: ARRAY [INTEGER] — Pivot row indices (partial pivoting)

Design by Contract

All features are fully specified with Design by Contract (DBC). Preconditions validate input correctness, postconditions guarantee output properties.

Common Preconditions

  • dimension_positive — Vector/matrix dimension must be > 0
  • index_valid — Array index must be in valid range
  • compatible_dimensions — Vectors/matrices must have matching dimensions for operations
  • square_matrix — Some operations (inverse, eigenvalue) require square matrices
  • well_conditioned — solve() requires condition_number ≤ 1e12

Common Postconditions

  • result_not_nan — Results are valid numbers (not NaN)
  • norm_non_negative — Vector norms >= 0
  • dimension_preserved — Operations preserve dimension
  • determinant_correct — Determinant matches mathematical definition
  • solution_satisfies_system — solve() result satisfies Ax=b
